# 🚀 הפעלת זיהוי AI אמיתי ב-FilaManager AI

## 🎯 הבעיה

כרגע האפליקציה רצה ב-**Demo Mode** - כשאתה מעלה תמונת סליל, היא לא מזהה אותו באמת אלא מציגה נתונים כלליים שצריך לערוך ידנית.

**בעיות ב-Demo Mode:**
- ❌ צבעים לא נכונים (תמיד אפור)
- ❌ יצרן כתוב "Unknown Brand"
- ❌ חומר כתוב "PLA" באופן אוטומטי
- ❌ שם צבע כתוב "Please Edit"

## ✅ הפתרון - הפעלת Gemini Vision AI

כדי שהאפליקציה **תזהה באמת את הסלילים מתמונות**, צריך להוסיף **Gemini API Key** (חינם!).

---

## 📋 שלבים להפעלת AI אמיתי

### שלב 1: קבלת API Key חינם מ-Google

1. **גש ל-Google AI Studio:**
   🔗 https://makersuite.google.com/app/apikey

2. **התחבר עם חשבון Google**

3. **לחץ על "Get API key"**

4. **העתק את המפתח** (מתחיל ב-`AIzaSy...`)

---

### שלב 2: הוספת ה-API Key לאפליקציה

#### אופציה 1: עריכת קובץ הקוד (מומלץ למפתחים)

1. **פתח את הקובץ:**
   ```
   lib/config/gemini_config.dart
   ```

2. **מצא את השורה:**
   ```dart
   static const String apiKey = 'YOUR_API_KEY_HERE';
   ```

3. **החלף ל:**
   ```dart
   static const String apiKey = 'AIzaSyAbc123...'; // המפתח שלך
   ```

4. **שמור את הקובץ**

5. **בנה APK מחדש:**
   ```bash
   flutter clean
   flutter pub get
   flutter build apk --release
   ```

#### אופציה 2: דרך מסך הגדרות באפליקציה (בקרוב)

- פתח את האפליקציה
- לחץ על **Profile** → **AI Setup**
- הדבק את המפתח ושמור
